<BlogSection>
  ## Essential Arduino Functions

  Let's understand the key functions we'll be using:

  <BlogList>
    <BlogListItem>
      **pinMode()** - Configures a pin as either INPUT or OUTPUT
      <BlogInlineCode>pinMode(7, OUTPUT);</BlogInlineCode>
    </BlogListItem>
    <BlogListItem>
      **digitalWrite()** - Sets a pin to HIGH (5V) or LOW (0V)
      <BlogInlineCode>digitalWrite(7, HIGH);</BlogInlineCode>
    </BlogListItem>
    <BlogListItem>
      **delay()** - Pauses the program for a specified time in milliseconds
      <BlogInlineCode>delay(1000); // Waits for 1 second</BlogInlineCode>
    </BlogListItem>
  </BlogList>

  <BlogCallout type="note">
    **Syntax Tip!**
    
    Remember to end each instruction with a semicolon (;) - it's like a period at the end of a sentence in Arduino code.
  </BlogCallout>
</BlogSection>

<BlogSection accent>
  ## The Blink Code

  Here's the complete code to make your LED blink:

  <BlogCode filename="LEDBlink.ino" language="cpp">
{`void setup() {
  // put your setup code here, to run once:
  pinMode(7, OUTPUT);
}

void loop() {
  // put your main code here, to run repeatedly:
  digitalWrite(7, HIGH);  // Turn the LED on
  delay(1000);           // Wait for 1 second
  digitalWrite(7, LOW);   // Turn the LED off
  delay(1000);           // Wait for 1 second
}`}
  </BlogCode>

  <BlogCallout type="tip">
    **Time Conversion:**
    
    <BlogList>
      <BlogListItem>**1000ms** - Equal to 1 second</BlogListItem>
      <BlogListItem>**500ms** - Equal to 0.5 seconds</BlogListItem>
      <BlogListItem>**2000ms** - Equal to 2 seconds</BlogListItem>
    </BlogList>
  </BlogCallout>
</BlogSection>

<BlogSection>
  ## Code Explanation

  Let's break down how the code works:

  <OrderedList>
    <OrderedListItem>
      **Setup Function**
      
      <BlogInlineCode>pinMode(7, OUTPUT);</BlogInlineCode> tells Arduino that pin 7 will be used to send signals (OUTPUT mode)
    </OrderedListItem>
    <OrderedListItem>
      **Loop Function - First Part**
      
      <BlogInlineCode>digitalWrite(7, HIGH);</BlogInlineCode> sends 5V to pin 7, turning the LED on
      <BlogInlineCode>delay(1000);</BlogInlineCode> makes the program wait for 1 second
    </OrderedListItem>
    <OrderedListItem>
      **Loop Function - Second Part**
      
      <BlogInlineCode>digitalWrite(7, LOW);</BlogInlineCode> sends 0V to pin 7, turning the LED off
      <BlogInlineCode>delay(1000);</BlogInlineCode> makes the program wait for another second
    </OrderedListItem>
    <OrderedListItem>
      **Continuous Operation**
      
      The loop function repeats indefinitely, creating the blinking effect
    </OrderedListItem>
  </OrderedList>

  <BlogCallout type="warning">
    **Common Mistakes to Avoid:**
    
    <BlogList>
      <BlogListItem>**Missing Semicolons** - Always end your code lines with a semicolon</BlogListItem>
      <BlogListItem>**State Confusion** - Double-check your HIGH and LOW states are correct</BlogListItem>
      <BlogListItem>**Pin Numbers** - Verify you're using the correct pin number in your code</BlogListItem>
      <BlogListItem>**Delay Values** - Make sure your timing values are in milliseconds</BlogListItem>
