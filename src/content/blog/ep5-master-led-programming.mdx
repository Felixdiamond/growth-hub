  ## Key Programming Concepts

  Let's review the essential functions we'll be using:

  <BlogList>
    <BlogListItem>
      **pinMode()** - Sets pin mode to INPUT or OUTPUT
      <BlogInlineCode>pinMode(7, OUTPUT);</BlogInlineCode>
    </BlogListItem>
    <BlogListItem>
      **digitalWrite()** - Controls component state (HIGH/LOW)
      <BlogInlineCode>digitalWrite(7, HIGH);</BlogInlineCode>
    </BlogListItem>
    <BlogListItem>
      **delay()** - Creates timing pauses in milliseconds
      <BlogInlineCode>delay(1000); // 1 second pause</BlogInlineCode>
    </BlogListItem>
  </BlogList>

  ## Advanced Blinking Pattern

  Here's our code for creating a more complex LED pattern:

  <BlogCode filename="AdvancedBlink.ino" language="cpp">
{`void setup() {
  // put your setup code here, to run once:
  pinMode(7, OUTPUT);
}

void loop() {
  // put your main code here, to run repeatedly:
  digitalWrite(7, LOW);   // LED off
  delay(1000);           // Wait 1 second
  digitalWrite(7, HIGH);  // LED on
  delay(1000);           // Wait 1 second

  digitalWrite(7, LOW);   // Second blink
  delay(1000);
  digitalWrite(7, HIGH);
  delay(1000);

  digitalWrite(7, LOW);   // Third blink
  delay(1000);
  digitalWrite(7, HIGH);
  delay(1000);

  digitalWrite(7, LOW);   // Extended off period
  delay(5000);           // Wait 5 seconds

  digitalWrite(7, HIGH);  // Extended on period
  delay(5000);           // Wait 5 seconds
}`}
  </BlogCode>
