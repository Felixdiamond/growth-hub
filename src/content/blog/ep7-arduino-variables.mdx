<BlogSection>
  ## Integer Variables (int)

  Integers are whole numbers without decimal points. They're perfect for counting, storing pin numbers, or managing simple numerical values.

  <BlogCode filename="IntegerExample.ino" language="cpp">
{`int num = 3;

void setup() {
  // Initialize serial communication
  Serial.begin(9600);
}

void loop() {
  // Print the value of our integer variable
  Serial.println(num);
}`}
  </BlogCode>

  <BlogCallout type="note">
    **About Integers:**
    
    <BlogList>
      <BlogListItem>**Declaration** - Use the <BlogInlineCode>int</BlogInlineCode> keyword</BlogListItem>
      <BlogListItem>**Values** - Whole numbers (no decimals)</BlogListItem>
      <BlogListItem>**Common Uses** - Counting, pin numbers, simple math</BlogListItem>
      <BlogListItem>**Memory** - Takes up 2 bytes of memory</BlogListItem>
    </BlogList>
  </BlogCallout>
</BlogSection>

<BlogSection accent>
  ## Best Practices

  Follow these guidelines for effective variable use:

  <BlogList>
    <BlogListItem>**Descriptive Names** - Use clear, meaningful variable names</BlogListItem>
    <BlogListItem>**Initialization** - Always give variables initial values</BlogListItem>
    <BlogListItem>**Scope** - Declare variables where they're needed</BlogListItem>
    <BlogListItem>**Data Types** - Choose the appropriate type for your data</BlogListItem>
  </BlogList>

  <BlogCallout type="warning">
    **Common Mistakes to Avoid:**
    
    <BlogList>
      <BlogListItem>**Wrong Data Type** - Using int for decimal values</BlogListItem>
      <BlogListItem>**Naming Conflicts** - Using reserved words as variable names</BlogListItem>
      <BlogListItem>**Uninitialized Variables** - Not giving variables starting values</BlogListItem>
      <BlogListItem>**Case Sensitivity** - Arduino is case-sensitive, so <BlogInlineCode>myVar</BlogInlineCode> and <BlogInlineCode>myvar</BlogInlineCode> are different</BlogListItem>
    </BlogList>
  </BlogCallout>
</BlogSection>

<BlogSection>
  ## Practical Examples

  Here's how to combine different variable types in a practical example:

  <BlogCode filename="CombinedExample.ino" language="cpp">
{`int sensorPin = 7;
float temperature = 25.5;
String unit = "Celsius";

void setup() {
  Serial.begin(9600);
}

void loop() {
  // Print formatted sensor reading
  Serial.print("Sensor on pin ");
  Serial.print(sensorPin);
  Serial.print(": ");
  Serial.print(temperature);
  Serial.print(" ");
  Serial.println(unit);
}`}
  </BlogCode>
